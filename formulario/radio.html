Como utilizar os Formularios de Botões de opção em HTML

Um botão de opção ou botão de rádio é um elemento de interface gráfica com dois estados: selecionado e não-selecionado, quando o usuário pressiona um botão de rato ou uma tecla como Espaço. Botões de opção usados num grupo para apresentar um conjunto limitado de escolhas que são mutuamente exclusivas.
